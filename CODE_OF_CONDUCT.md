# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and collaborate in good faith.
- Provide actionable technical feedback.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Publishing private information without permission.
- Disruptive behavior that blocks healthy collaboration.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, wiki edits, issues, and other contributions that violate this Code of Conduct.
